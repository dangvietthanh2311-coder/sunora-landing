Sunora: Desktop và Mobile cạnh nhau

Cùng một file index.html, hai khung mô phỏng hai thiết bị. Dùng để trình bày trực tiếp khác biệt responsive trước lớp.

Desktop (khung 1280px, thu nhỏ 58%)
Mobile (khung 390px)

Gợi ý khi trình bày: chỉ vào từng khác biệt theo bảng trong responsive-notes.md: menu ngang so với hamburger, hero 2 cột ảnh ngang so với xếp dọc ảnh đứng, lưới sản phẩm 3 cột so với 1 cột, cỡ tiêu đề lớn so với nhỏ, nút trong hàng so với nút full width, và footer nhiều cột so với xếp dọc. Nếu khung desktop bị cắt, mở thẳng index.html rồi kéo rộng hoặc thu hẹp cửa sổ trình duyệt để xem trang tự đổi bố cục.
